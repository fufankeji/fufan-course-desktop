# 课程知识库 FAQ

## 学员应该先看视频还是先问知识库？

建议先看当前模块的视频，再问知识库。知识库适合复习、查漏补缺、让 Agent 生成执行清单。

## 为什么不用一上来就做向量数据库？

课程资料是结构化资产，先编译成 LLM Wiki 可以得到更稳定的概念页、项目页和 FAQ 页。等规模变大后，再加 embedding 或 BM25。

## Agent 回答和老师讲的不一样怎么办？

要求 Agent 引用知识库页面。如果没有引用，就让它重新基于具体页面回答。

## 如何定位报错？

把报错、当前模块和项目目标一起问知识库。优先查工具页、FAQ 页和项目手册。
