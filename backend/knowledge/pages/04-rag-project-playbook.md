# 企业级 RAG 项目实战手册

这个项目是课程的第一个旗舰项目。目标不是只做一个聊天框，而是做一个可以展示给业务方看的高精度知识库问答系统。

## 项目模块

- 数据导入：支持 Markdown、HTML、PDF 和 Notebook 转换后的 Markdown。
- 知识页编译：把原始资料转成概念页、项目页、FAQ 页。
- 检索问答：先关键词和标签检索，再组合证据生成回答。
- 引用来源：每个回答必须列出引用页，方便学员回看课程。
- 前端体验：左侧目录、中间知识页、右侧问答。

## 推荐学习顺序

1. 先学 [[RAG 核心概念地图]]。
2. 再实现最小检索问答 API。
3. 然后补充评估集和错误样例。
4. 最后把项目打包成交付产品。

## 可验收标准

- 输入“RAG 评估怎么做”能返回评估相关页面。
- 输入“我想做企业知识库问答”能给出项目路线。
- 回答里必须包含来源引用。
