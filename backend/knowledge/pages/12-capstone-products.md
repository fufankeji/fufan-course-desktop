# 课程项目产品包

项目产品是课程升级后的第四类交付。它不能只是代码压缩包，而应该是“拆包即用”的学习和展示资产。

## 标准结构

- `README.md`：项目定位、启动方式、演示问题。
- `backend/`：后端 API 和测试。
- `frontend/`：前端界面。
- `knowledge/`：项目知识库或示例数据。
- `skills/`：可选 Agent Skill。
- `evals/`：评估集和回归脚本。

## 四个旗舰方向

- 企业级 RAG 问答系统。
- 文档审查 Agent。
- Claude Code/OpenClaw 自动化研发项目。
- 数据分析或 NL2SQL Agent。

## 展示方式

学员求职或转型时，可以展示运行地址、项目架构、评估报告和 Agent 协作过程。
