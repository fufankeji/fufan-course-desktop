# 如何让你的 Agent 学会这门课

课程升级后的核心话术是：不仅教会你，还教会你的 Agent。

## 学习闭环

1. 学员观看视频，理解老师的项目思路。
2. Agent 读取对应 Wiki 页面，复述知识结构。
3. 学员让 Agent 生成任务清单和检查清单。
4. Agent 调用 Skill 完成项目局部任务。
5. 学员用项目产品和测试结果验收。

## 推荐提问

- “请根据这节课知识库，给我生成三步学习路线。”
- “请把这个项目拆成可以交给 Claude Code 的任务。”
- “我遇到这个报错，应该回看哪一页？”

## 交付价值

这让课程从单向视频变成“人和 Agent 共同学习的系统”。学员不只是看完内容，而是拥有一套可持续调用的知识资产。
